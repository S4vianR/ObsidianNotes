# Definiciones

## Símbolo
**Símbolo**: Representación de algo.
Ejemplos de símbolos:
- Letras = { a,b,c,d,e,f,g, ... }
- Símbolos = { !,",#,$,%,&, ... }
## Alfabeto
**Alfabeto**: Conjunto no vacío y finito de símbolos
Ejemplos de alfabetos:
- Alfabeto del español.
- Alfabeto del griego.
